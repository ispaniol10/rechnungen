⚙️ Baserow konfigurieren

Trage deinen BASEROW_TOKEN und die BASEROW_TABLE_ID ein (ganz oben im <script>-Block).
Erstelle außerdem folgende Tabelle in Supabase → SQL Editor:

CREATE TABLE rechnungen (id UUID PRIMARY KEY DEFAULT gen_random_uuid(), year INTEGER NOT NULL, name TEXT NOT NULL, re_nr TEXT, betrag DECIMAL(10,2), re_datum DATE, eingereicht DATE, bezahlt DATE, person TEXT, sort_idx INTEGER DEFAULT 0, created_at TIMESTAMPTZ DEFAULT NOW());
ALTER TABLE rechnungen ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all" ON rechnungen FOR ALL TO anon USING (true) WITH CHECK (true);

Bis dahin werden Daten lokal gespeichert (localStorage).
